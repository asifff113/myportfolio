# App Phase Snapshot

**Date:** November 17, 2025

## Current Phase Overview

- Key directories: `src/app`, `src/components`, `src/hooks`, `src/lib`, `src/styles`.
- Main features implemented:
  - Public pages: Home, Blog, Contact
  - Admin section: Personal Info, Projects

- Firebase setup and Firestore structure documented
- ESLint and Tailwind configured
